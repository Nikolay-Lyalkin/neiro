Мини-приложение предназначенное для хранения координат геометрических фигур.

# Запуск:
1. Скопировать проект
2. Запустить файл - docker-compose.yml из корня проекта - "docker-compose up -d --build"
3. Подключиться к контейнеру - "docker compose exec web bash"
4. Запустить код в командной строке - "poetry run python src/main.py"
